## Prinsip

Bangun fondasi penuh sejak awal; aktifkan fitur bertahap. Urutan menekankan **konteks & kualitas** sebelum perluasan.

## Tahapan (indikatif)

### Tahap 0 — Fondasi
- Setup arsitektur core modular, DB/vector, event bus.
- Data pipeline dasar (market, news) → knowledge base.

### Tahap 1 — MVP
- Market Intelligence Layer (Market Brief harian).
- Investment Committee inti + Decision/Confidence/Explainability.
- Telegram bot: `/brief`, `/analyze`, watchlist.
- Evaluation framework awal + golden set.

### Tahap 2 — Kualitas & Kepercayaan
- Social & fundamental pipeline matang.
- Alert event-driven.
- Backtesting + kalibrasi confidence + HITL.

### Tahap 3 — Ekspansi Interface
- Dashboard web (core sama), API internal.
- Perluasan cakupan emiten.

### Tahap 4 — Platform (AI Investment OS)
- Simulasi portfolio, continuous learning rutin.
- Kesiapan adapter multi-market (opsional).
